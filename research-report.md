### 2. [sharp](https://github.com/lovell/sharp)

> High performance Node.js image processing — the fastest module to resize JPEG, PNG, WebP, AVIF and TIFF images

| Metric | Value |
|--------|-------|
| **Stars** | 32,056 |
| **Last Updated** | 2026-03-23 |
| **License** | Apache-2.0 |

**Why it's relevant:** The `assets/` directory contains 210 MB of unoptimized JPEG and PNG images. Many merch images are full-resolution product shots served directly. **What it replaces/enhances:** Currently images are served as-is in their original format and resolution. A sharp-based build script would generate optimized variants and could be paired with `<picture>` elements for modern format delivery.
